# Rosalind Bioinformatics Exercises

This repository contains my solutions to bioinformatics problems from the Rosalind platform (http://rosalind.info/), implemented in Python.

## About
Rosalind is an online platform that provides computational biology problems to help develop skills in bioinformatics and programming.  
Through these exercises, I am strengthening my ability to solve biological problems using code and algorithmic thinking.

## Topics Covered
- DNA and RNA sequence analysis  
- String manipulation and pattern matching  
- Basic bioinformatics algorithms  
- Data handling and processing in Python  

## Technologies
- Python  

## Objectives
- Improve problem-solving and logical thinking  
- Apply programming to biological data  
- Build a foundation in bioinformatics and computational biology  

## Structure
Each file corresponds to a specific Rosalind problem and contains a Python solution.

## Progress
This repository is continuously updated as I complete new exercises.
